O que é HTML?

(Inicio)

HTML, que significa Linguagem de Marcação de Hipertexto (em inglês, Hypertext Markup Language), é a espinha dorsal da criação de páginas web. Pense no HTML como a estrutura básica de uma casa. Assim como uma casa tem paredes, um telhado, janelas e portas, uma página web tem elementos como texto, imagens, links e formulários. A principal função do HTML é estruturar o conteúdo da página. Ele utiliza "tags" para envolver diferentes partes do conteúdo e dizer ao navegador como essas partes devem ser exibidas. As tags são como instruções para o navegador sobre como organizar e apresentar as informações.

HTML é Linguegem de porogamação?

(Inicio)

Bem, a galera costuma dizer que HTML não é uma linguagem de programação porque ele não é muito esperto. Sabe, uma linguagem de programação é como um cérebro que pode fazer cálculos, tomar decisões e realizar tarefas complicadas. O HTML não é assim tão inteligente. O HTML é mais como um organizador de informações. Ele diz para o navegador como mostrar as coisas na tela, como se fosse o diretor de um espetáculo teatral. Ele fala: "Ei, este pedaço é um título, aquele ali é um parágrafo, e aqui tem uma imagem para exibir". Mas não pode fazer contas, não pode criar funções ou tomar decisões sozinho. Então, enquanto as linguagens de programação têm esse poder cerebral para fazer cálculos e decisões, o HTML é mais como um assistente que ajuda a organizar e mostrar as coisas bonitinhas na internet. É por isso que a galera fala que HTML não é uma linguagem de programação, mas sim uma linguagem de marcação. Ele marca como as coisas devem aparecer, mas não tem o poder de pensar muito.

O que são TAGS?

(Inicio)

Certo, imagine que HTML é como um chef que está preparando uma receita para o navegador exibir. As tags seriam como as instruções específicas que o chef dá para cada ingrediente e passo na receita. Então, uma tag em HTML é como um lembrete para o navegador sobre o que fazer com uma parte específica da página. Elas são essencialmente comandos que dizem: "Ei, aqui começa um título", ou "Aqui está um parágrafo de texto", ou até mesmo "Mostre esta imagem aqui". Por exemplo, se você quiser adicionar um título à sua página, você usaria a tag H1 Se quiser um parágrafo, você usaria a tag P. E para uma imagem, seria IMG. Então, as tags são como as instruções do chef na receita. Elas dizem ao navegador como organizar e mostrar cada pedacinho da página web, para que tudo fique no lugar certo e pareça legal quando alguém acessar o site.

Alguns exemplos de TAGS

(Inicio)

No HTML temos diversas TAGS, aqui está o exemplo de algumas delas. Temos as tags utilizadas para Titulos como H1, H2, H3, H4... essas tags são como titulos de um livro, sendo H1 o titulo mais o importante e H6 o menos. Use-as para organizar e dar destaque aos diferentes tipos de informação.

Tag P. Essa tag é como uma caixa para colocar seu texto. Cada vez que você quiser um novo bloco de texto, use P.

Tag A. Essa tag cria links para outras páginas. É como uma porta que leva você a diferentes lugares na web.

Tag IMG Quer colocar uma foto na sua página? Use IMG. É como pendurar um quadro na parede da sua página web.

Tags UL, OL, Li, são Tags de listagem. Use essas tags para criar listas. Ul para lista não ordenadas(como uma lista de compras) e Ol para listas ordenadas como a de uma receita de bolo. Ja a tag Li faz uma listagem de itens.
